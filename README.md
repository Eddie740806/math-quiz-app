# 📐 國小數學題庫系統

支點教育 K12 團隊專屬練習平台

## 功能特色

- ✅ 用戶註冊/登入系統
- ✅ 五年級數學練習（25題）
- ✅ 六年級數學練習（25題）
- ✅ 即時答題與批改
- ✅ 錯題本記錄
- ✅ 學習進度追蹤

## 技術架構

- **前端**: Next.js 16 + React + TypeScript
- **樣式**: Tailwind CSS
- **資料儲存**: localStorage（可擴展至資料庫）
- **部署**: Vercel（免費）

## 本地開發

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 建置生產版本
npm run build

# 啟動生產版本
npm start
```

## 訪問地址

- 開發模式: http://localhost:3000
- 生產模式: http://localhost:3000

## 頁面結構

| 路徑 | 功能 |
|------|------|
| `/` | 首頁（年級選擇） |
| `/register` | 用戶註冊 |
| `/login` | 用戶登入 |
| `/quiz?grade=5` | 五年級刷題 |
| `/quiz?grade=6` | 六年級刷題 |
| `/wrong-answers` | 錯題本 |

## 題庫內容

### 五年級（25題）
- 分數與小數
- 分數加減乘除
- 面積計算
- 因數與倍數
- 質數與合數
- 周長計算
- 小數乘除法

### 六年級（25題）
- 正負數運算
- 代數方程式
- 圓周長與面積
- 體積計算
- 比與比值
- 百分率
- 速率問題
